# @amlplugins/groq-embeddings

Native `groq-sdk` SDK re-exported under the `@amlplugins` namespace.

> Groq Embeddings — fast embedding generation for retrieval and clustering pipelines.

## Install

```bash
npm install @amlplugins/groq-embeddings
```

## Use

```ts
import { /* named exports from groq-sdk */ } from "@amlplugins/groq-embeddings";
// or
import sdk from "@amlplugins/groq-embeddings";
```

Built and published by GitHub Actions on every push to `main`.

Part of the amlplugins family — see https://github.com/amlplugins.
